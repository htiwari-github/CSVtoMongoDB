# CSVtoMongoDB

Export your CSV file to Mongo Db with this Python Code 

```python
    mongodb = MongoDB(dBName = 'Dataset', collectionName='EnergyConsumption')
    mongodb.InsertData(path="AEP_hourly.csv")

```
